h1. FxJsJUnit

FxJsJUnit is a JUnit extention for unit testing javascript on webkit offered by JavaFX2.0's WebView.

h2. Required Environment

* Oracle jdk1.7 or higher
* JavaFX 2.0 or higher
* JUnit 4.10 or higher
* Tested Operating System
** Windows7
** Mac OS X 10.7

h2. For build from source

* Gradle milestone-8 or later
* Oracle jdk 1.7 or higher
* JavaFX 2.0 or higher
